# Keycard Support-Escalation Workshop

This repo is a hands-on workshop: you help an attendee build a support-escalation MCP
server and secure it with Keycard, one chapter at a time.

**Before helping, read the workshop helper skill and follow it:**
`.claude/skills/keycard-workshop/SKILL.md` (also at `.agents/skills/keycard-workshop/SKILL.md`).
It defines how to orient, which chapter to build, and the guardrails — most importantly:
build one chapter at a time, in order, never run ahead, and treat the docs as the source
of truth.

Key layout:

- You work in **`support-escalation/`** for the entire workshop (the Chapter 0 baseline).
- **`mcp-server/`** is the finished server (Chapters 6–7), a read-only reference.
- The workshop guide is the hosted docs site at **https://workshop.keycard.ai** — start at
  the prerequisites, then work through the chapters in order.
